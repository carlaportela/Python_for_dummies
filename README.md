## 📘 Descripción

Este repositorio contiene ejercicios, ejemplos prácticos y apuntes del aprendizaje de **Python desde cero**, siguiendo el curso de la plataforma  de **Youtube** del canal **Píldoras Informáticas**.

El objetivo del repositorio es reforzar conceptos básicos y avanzados de programación en Python mediante práctica constante y ejercicios progresivos.

---

## 🎯 Objetivos del repositorio

- Aprender fundamentos del lenguaje Python  
- Practicar lógica de programación mediante ejercicios  
- Comprender estructuras de control, funciones y POO  
- Desarrollar habilidades de resolución de problemas  
- Construir una base sólida como desarrolladora  

---

## 📚 Contenido del curso

### 🧩 Nivel básico

- Sintaxis básica de Python  
- Variables y tipos de datos  
- Operadores  
- Condicionales (`if`, `else`, `elif`)  
- Bucles (`for`, `while`)  

---

### 🧩 Nivel intermedio

- Funciones  
- Listas, tuplas y diccionarios  
- Manejo de errores  
- Módulos y paquetes  

---

### 🧩 Nivel avanzado

- Programación orientada a objetos (POO)  
- Herencia y polimorfismo  
- Manejo de archivos  
- Buenas prácticas de código  

---

## 🧠 Metodología

Cada tema incluye:

- 📌 Teoría resumida  
- 💻 Ejercicios prácticos  
- 🧪 Resolución paso a paso  
- 🔁 Repetición para consolidación
